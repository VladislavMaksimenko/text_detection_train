1). Файлы для обучения поместить в директорию data -> train -> class1; data -> train -> class2; .... data -> train -> classN
2). Файлы для проверки обучения поместить в директорию data -> validation -> class1; data -> validation -> class2; .... data -> validation -> classN,
    то есть, структура директорий train и validation должна быть одинаковой
3). В каждой директории равное количество изображений. Все изображения имеют расширение .jpg.
4). Конфигурацию нейронки подбирать по ситуации, в комментариях всё подробно описано.
